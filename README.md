# 📚 Digital Study Assistant

## Overview
Digital Study Assistant is a simple student productivity application developed using Python and HTML. The project is designed to help students organize their academic activities in one place.

## Features
- 📝 Manage study notes
- ✅ Create and manage daily tasks
- ⏱️ Built-in study timer
- 🎓 GPA Calculator
- 💬 Motivational quotes for students

## Technologies Used
- Python
- HTML

## Purpose
The purpose of this project is to improve students' productivity by providing useful study tools in a simple interface.
